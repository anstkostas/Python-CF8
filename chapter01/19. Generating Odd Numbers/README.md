## 3. Key Concepts

- **range(start, stop, step)**: Built-in function that generates an immutable sequence of numbers from `start` (inclusive) to `stop` (exclusive), incremented by `step`.
- **Start**: The first value in the sequence (default is 0).
- **Stop**: The sequence ends before this value.
- **Step**: The difference between each number in the sequence (default is 1).
- **Odd Numbers**: Integers not divisible by 2; can be generated by starting at an odd number and stepping by 2.

## 5. Code Demos

```
# Python 2 vs Python 3 range functionality discussion
# Python 2: `range()` returned a list, not memory efficient
# Python 3: `range()` returns a range object, which is much more memory efficient

# Print odd numbers from 1 to 21 using a traditional Java-style approach in Python
print("Java style")
for i in range(22):
    if i % 2 != 0:
        print(i, end=' ')
print()

# Print odd numbers from 1 to 21 using Python's efficient range with step parameter
print("Python style")
for i in range(1, 22, 2):
    print(i, end=' ')
print()
```

### 1. (Level 1, MCQ)
Which of the following will generate the sequence `3, 5, 7, 9`?

a) `range(3, 10, 2)`  
b) `range(3, 9, 2)`  
c) `range(3, 10, 3)`  
d) `range(3, 11, 2)`

<details>
<summary>Solution</summary>

**Answer:** a) `range(3, 10, 2)` and d) `range(3, 11, 2)`  
Both generate 3, 5, 7, 9.  
- `range(3, 10, 2)` gives 3, 5, 7, 9  
- `range(3, 11, 2)` gives 3, 5, 7, 9 (and would include 11 if stop were inclusive, but stop is exclusive)

### 4. (Level 2, Coding)
Given two variables, `start = 8` and `stop = 20`, write code to print all odd numbers between `start` and `stop` (inclusive), regardless of whether `start` is odd or even.

<details>
<summary>Solution</summary>

```python
start = 8
stop = 20
# If start is even, make it odd
if start % 2 == 0:
    start += 1
for i in range(start, stop + 1, 2):
    print(i, end=' ')
# Output: 9 11 13 15 17 19 
```
</details>

### 5. (Level 3, Coding)
Write a function `odd_numbers_between(a, b)` that returns a list of all odd numbers between `a` and `b` (inclusive), where `a` and `b` can be in any order (i.e., a may be less than or greater than b).

<details>
<summary>Solution</summary>

```python
def odd_numbers_between(a, b):
    # Ensure a <= b
    start, stop = min(a, b), max(a, b)
    # Make start odd if it's even
    if start % 2 == 0:
        start += 1
    # range is exclusive, so stop+1
    return list(range(start, stop + 1, 2))

# Example usage:
print(odd_numbers_between(12, 21))  # Output: [13, 15, 17, 19, 21]
print(odd_numbers_between(21, 12))  # Output: [13, 15, 17, 19, 21]
```
</details>
